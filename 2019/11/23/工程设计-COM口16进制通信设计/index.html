3.4 对接收数据进行的异常处理

关于异常处理与提取,接收数据可能是20个byte,需要在这20个byte中提取有效的长度。

  1. 这个有效长度必须满足收尾都是0x64
  2. 这个有效长度最短长度不得少于8个byte
  3. 这个有效长度的数据必须符合校验和的计算方式
  4. 满足上面3条即可提取这个数据帧
  5. 注意:在真正想要的数据帧之间是没有0x64的,这种情况先假设它不存在(后面会根据校验和的方式来处理)
